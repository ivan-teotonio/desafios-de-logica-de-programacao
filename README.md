﻿🟢 Dia 1 — Exercícios de Lógica (Junior)

Resolva sem pensar em código primeiro. Pense no passo a passo da solução.

1️⃣ Par ou Ímpar

Receba um número e determine se ele é:

par

ímpar

2️⃣ Maior de três números

Você recebe 3 números diferentes.

Descubra qual é o maior deles.

3️⃣ Contagem até N

Receba um número N.

Liste todos os números de 1 até N.

Exemplo:

N = 5

Resultado esperado:

1, 2, 3, 4, 5

4️⃣ Soma de números

Você recebe uma lista de números.

Calcule a soma de todos eles.

5️⃣ Tabuada

Receba um número e mostre a tabuada dele de 1 até 10.

6️⃣ Fatorial

Receba um número N.

Calcule o fatorial dele.

Exemplo:

5 → 5 × 4 × 3 × 2 × 1

7️⃣ Número primo

Receba um número.

Descubra se ele é primo ou não.

8️⃣ Inverter palavra

Receba uma palavra.

Retorne a palavra invertida.

Exemplo:

casa → asac

9️⃣ Contar vogais

Receba uma palavra.

Conte quantas vogais existem nela.

🔟 Maior número de uma lista

Receba uma lista de números.

Descubra qual é o maior número da lista.
